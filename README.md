# Earthquake Risk Analysis using QGIS

## Project Overview
This project analyzes earthquake data to identify high-risk zones using GIS techniques.

## Tools Used
- QGIS
- Vector data (points and polygons)

## Key Analysis
- Buffer analysis around epicenters
- Spatial join
- Heatmap analysis
- Thematic mapping

## Outcome
The project highlights earthquake-prone areas useful for disaster management and planning.
